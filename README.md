# Emberglow Dotfiles — Omarchy + Hyprland

A warm, amber-toned desktop setup for [Omarchy](https://omarchy.org/) (Arch Linux + Hyprland).

## Features

- **Emberglow theme** — custom amber/warm color palette
- **Floating waybar** — glassmorphism style with pill-shaped workspaces
- **Smart keybindings** — web apps, tmux, music, screenshots
- **Custom widgets** — weather, system stats, GPU monitor
- **Full visual consistency** — btop, neovim, VS Code, terminals all themed

## Installation

```bash
git clone https://github.com/vaibhavsingh-shekhawat/custom-omarchy-dotfiles.git
cd custom-omarchy-dotfiles
chmod +x install.sh
./install.sh
```

Then reboot or restart Hyprland.# custom-omarchy-dotfiles
